# Weather Today ☀️

A clean, simple weather app that shows current temperature, feels-like, wind, humidity, and weather condition for any city.

Built with pure **HTML + CSS + JavaScript** — no frameworks, no API key needed!  
Uses the free [Open-Meteo](https://open-meteo.com/) weather API.

## Features
- Search any city name  
- Shows: temperature (°C), weather description, wind speed, humidity, feels-like  
- Beautiful nature background  
- Works on mobile and desktop  
- Default starts with Mumbai weather  

## Technologies Used
- HTML5  
- CSS3 (simple & responsive)  
- Vanilla JavaScript (Fetch API)  
- Font Awesome icons  
- Open-Meteo Geocoding + Forecast API (free & no signup)
